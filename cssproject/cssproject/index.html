CSS Challenge

In this project you will start off with some ready-made HTML, and write some CSS to solve the problems set out in the questions below.

Grab the prepared HTML file and follow the project instructions to set up your files for this project. Ready to get started? Here you go.

Part 1 Tasks

The numbered items below are the tasks you need to complete in the first part of this project. These are simple exercises designed to test your knowledge of common CSS selectors and text related properties. You will need to target specified elements with an assortment of selectors: element type selectors, grouped selectors, contextual selectors (descendant, child, and adjacent sibling selectors), class selectors, and id selectors.

Each one of these should take you only a minute or so to apply, once you have your style sheet set up. If you get stuck, refer back to chapters 11 & 12 in your book.

  1. Write a rule that will globally apply an appropriate sans-serif font stack default for every element on the page, and globally set the default base font size and spacing between lines for all child elements.
  2. Override the previous font setting just for the headingsâ€”write a rule that will make all heading levels 1 and 2 use an appropriate serif font stack of your choice.
  3. Select an appropriate color scheme for this site; two or three colors should be sufficient. Write a rule to apply a relative font size (other than default) and appropriate text color only for the heading 1 element.
  4. Write a rule to apply a relative font size (other than default) and appropriate text color for all heading 2 elements. Just be sure that headings are larger than paragraphs, and heading 2 elements are smaller than heading 1.
  5. Write a rule to apply an appropriate text color and increase the spacing between lines for all paragraph elements.
  6. Write a rule to apply italic and appropriate text color just for the quotation text (should not apply to cited source).
  7. Write a rule to apply a relative font size (smaller than default) and appropriate text color only to the text in the page footer.
  8. Write a rule to apply an appropriate Web font (display category) for only the heading 1 element, center align it, and apply decorative properties of your choice (letter-spacing, text-shadow, text-transform but not text-decoration).
  9. Write a rule to indent the first line of each paragraph that directly follows a level 2 heading.
  10. Write a rule to apply special emphasis to all text elements that belong to the message class. The properties are your choice.

Part 2 Tasks

The CSS Challenge continues with exercises to test your knowledge of working with colors, backgrounds, and element boxes. The numbered items below are the tasks you need to complete in the second part of this project.

If you get stuck, refer back to chapters 13 & 14 in your book.

  1. Write a rule to set the background color of the page using either RGB or hexadecimal color values, set the page width to less than 100%, and set the page margins to auto.
  2. Update all color properties with color names to use RGB or hexadecimal values in all existing style rules.
  3. Write a rule to apply visual interest to all elements that belong to the resources id (div element). Set a background color (use RGB or hexadecimal value), appropriate width, apply appropriate padding and margins, and a border (your choice of color, style, width, and sides). If appropriate visually for your design, you may apply rounded borders and/or apply a drop shadow.
  4. Write rules for the five link states so they are a different color than the rest of the text, and when you mouse over them, the underline either appears or disappears. The styles for the link and hover states should be visually different, and remember that order matters.
  5. Write a rule to apply a bottom border (your choice of style, width, and color) just to the abbr elements to style them consistently in all browsers.
  6. To keep the level 2 headings closer to the paragraph that follows it and farther away from the paragraph that precedes it, write a rule to set bottom margin of level 2 headings to zero, and write a rule to set top margin of each paragraph that directly follows a level 2 heading to zero.
  7. To increase the space between the list item elements, write a rule to add bottom margin to the list items.
  8. Write a rule to apply an appropriate background image to the page header element. Remember that header element height is determined by height of child elements inside the header. Increasing the font size and/or top and bottom padding in h1 element rule will increase the height of the header, exposing more area of background image.
  9. Write a rule to apply a background gradient to the page footer element. Gradient should not apply to footer inside blockquote element. If necessary, write another rule to set appropriate text color for the paragraph text inside the page footer.
  10. Modify all the style rules to use shorthand wherever appropriate.
